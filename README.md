# DataAnalysis_SemanticWeb
The aim of this project is to find out which is the most efficient energy source for electricity production and reduction of air pollution. The 3 datasets used in this project are about US electricity generation, consumption and pollution for the years 1990 â€“ 2016 at Federal level. The datasets contain details regarding the energy sources used for the generation or production of electricity like Coal, Natural Gas, Petroleum and Geothermal. The pollution dataset consists of data about 3 different pollutants: CO2, SO2 and NOX which are produced during the usage of energy source to generate electricity.
